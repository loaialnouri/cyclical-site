# Cyclical

Privacy policy and support page for the Cyclical iOS app.

**Live site:** [loaialnouri.github.io/cyclical-site](https://loaialnouri.github.io/cyclical-site/)

## What is Cyclical?

Cyclical is a cycling computer built for iPhone, featuring a customisable dashboard, BLE sensor support, turn-by-turn navigation, and ride analytics — designed with Apple's Liquid Glass for iOS 26.
